# ✈️ Flight Radar Clone

Hello! In this project, I developed a **Flight Radar Clone** application.  
You can observe air traffic in detail with this app that simulates real-time air traffic and allows you to track flights on an interactive map. 🛫🛬

## 🛠️ Technologies and Libraries Used

- ⚛️ **React** – Component-based modern frontend library
- 🛠️ **Redux Toolkit** – Modern state management solution
- 🌐 **Axios** – For API requests
- 🎨 **Bootstrap** – For responsive UI design
- 🗺️ **Leaflet** + **React-Leaflet** – Flight map and marker display
- 🔄 **Leaflet-RotatedMarker** – Shows direction of airplanes
- 🔘 **React-Paginate** – Pagination support
- 🎯 **React-Router-Dom** – Routing and navigation
- 🔗 **React-Icons** – Icon support
- 🖼️ **@splidejs/react-splide** – Image slider/carousel component

---

## 🎨 Features

- ✔️ Real-time flight data with API integration
- ✔️ Interactive map displaying flight positions
- ✔️ Rotated airplane markers based on direction
- ✔️ Flight list view with pagination
- ✔️ Responsive and mobile-friendly layout
